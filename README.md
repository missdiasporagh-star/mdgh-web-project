# Miss Diaspora Ghana (MDGH) Website

A beautiful, modern pageantry website for Miss Diaspora Ghana, celebrating beauty, culture, and impact across the diaspora.

## About Miss Diaspora Ghana

Miss Diaspora Ghana (MDGH) is an initiative of Nubian Crown Company Limited, dedicated to empowering young women of African descent to lead, serve, and connect Ghana with its global diaspora.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI/UX**: Beautiful, elegant design with smooth animations
- **Video Hero Section**: Engaging hero section with background video
- **Sponsorship Tiers**: Detailed sponsorship information with benefits
- **Interactive Elements**: Scroll animations, counters, and smooth scrolling
- **Contact Form**: Easy-to-use contact form for inquiries
- **Performance Optimized**: Fast loading times and optimized assets

## Website Structure

- **Hero Section**: Video background with call-to-action buttons
- **Three Pillars**: Cultural Identity, Leadership Training, Social Impact
- **About Section**: Information about MDGH and Nubian Crown Company Limited
- **Objectives**: Six key objectives of the organization
- **Programs**: Six main programs and initiatives
- **Statistics**: Impact metrics and achievements
- **Sponsorship**: Four-tier sponsorship structure with benefits
- **Contact**: Contact information and inquiry form

## Setup Instructions

1. Clone or download this repository
2. Ensure all image files are in the same directory:
   - `mdgh intro vid.mp4` (hero video)
   - `culture.png`
   - `leadership.png`
   - `social impact.png`
   - `a-photo-of-a-confident-elegant-ghanaian.jpeg`
3. Open `index.html` in a web browser
4. For production, serve through a web server for best performance

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #F8B92F;
    --secondary-color: #000000;
    /* ... */
}
```

### Content
Edit content directly in `index.html` sections

### Styling
Modify styles in `styles.css` as needed
